# CodvedaInternTask3-Basic
Task 3: Introduction to JavaScript 

## Project Title
Introduction to JavaScript – Basic Interactivity

## Description
This project is created as **Task 3 (Basic)** of the **Web Development Internship at Codveda Technologies**.  
The objective of this task is to understand and implement **basic JavaScript interactivity** on a webpage.

## Objectives Covered
- Create interactive elements using JavaScript
- Implement dropdown navigation
- Open and close modal popup
- Validate form inputs using JavaScript
- Use JavaScript functions and event listeners

## Features
- Dropdown menu with clickable navigation
- Content sections open on user interaction
- Modal popup with close functionality
- Contact form with basic validation
- Clean and simple user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How to Run the Project
1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in any web browser or use Live Server
